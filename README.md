# Thunderbolt Dispatch OS

Thunderbolt Dispatch OS is an end-to-end platform for managing trucking operations.

## Structure

- `/frontend`: React Native / Expo project for Web (Admin/Customer) and Mobile (Driver).
- `/backend`: NestJS Node.js application.
- `/database`: SQL Server schema and migrations.
- `/ci-cd`: GitHub Actions workflows.
